# 更新日志

## v1.1.0 - 2024-03-12

### 🎉 新增功能
- **对话上下文记忆**: AI现在能记住之前的对话内容，实现连续对话
- **智能上下文管理**: 自动保存和清理对话历史
- **多场景支持**: 支持私聊和群聊的独立上下文管理
- **改进的视频处理**: 智能检测视频文件并提供友好的交互提示
- **手动管理命令**: 
  - `清除记忆` / `clear memory` - 清除当前对话记忆
  - `查看记忆` / `show memory` - 查看当前上下文状态

### ⚙️ 新增配置项
- `enableContext`: 启用上下文功能（默认: true）
- `maxContextMessages`: 最大上下文消息数（默认: 10）
- `contextTimeout`: 上下文超时时间，分钟（默认: 30）
- `separateGroupContext`: 群聊独立上下文（默认: true）

### 🔧 技术改进
- 优化内存使用，定期清理过期上下文
- 改进多媒体消息处理，避免上下文中存储大文件
- 增强视频文件检测和用户友好的提示信息
- 增强错误处理和日志记录

### 🐛 问题修复
- 修复视频文件无法被AI识别的问题
- 改进CQ码解析的稳定性
- 优化多媒体内容的处理逻辑

### 📝 使用说明
1. 更新插件后，上下文功能默认启用
2. 视频文件现在会被正确检测并提供交互提示
3. 可通过配置界面调整上下文参数
4. 使用管理命令控制对话记忆
5. 详细使用方法请参考 `上下文功能使用指南.md`
